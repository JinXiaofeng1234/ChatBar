# ChatBar
ChatBar是一款以python为底层语言，pygame为GUI第三方库的开源大模型角色扮演对话项目，在大模型选择上目前支持文心一言，通义千问，ollama qwen2.5 14b 6p，网络搜索基于kimi大模型，支持知识库接入。 注意：项目的上手门槛可能比较高，请谨慎下载源码
api.txt记录了要申请的api，此外role_cards, bgm，imgs/black_room中的立绘，音频，图像，需要自己添置，因为我使用的素材未经授权，所以没有上传，请先行删除其文件夹中的txt文件，注意一定要安装ollama
